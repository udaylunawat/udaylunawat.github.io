Overview

PDF2Podcast is a Google NotebookLM-style text-to-speech application that transforms PDF documents into conversational podcasts. It leverages Kokoro, a lightweight open-weight TTS model with 82M parameters, and Kyutai (Moshi), a new open-source real-time TTS model (launched July 2025), to generate natural-sounding audio. A clean, user-friendly interface is built using Gradio.

Features

  • PDF to Audio Conversion: Upload any PDF and convert it into a podcast-style conversation.
  • Multiple Voice Options: Assign different voices for multiple speakers.
  • API Flexibility: Supports OpenAI, OpenRouter, and Kyutai (Moshi) APIs.
  • Parallel Processing: Utilizes all CPU cores for faster audio generation.
  • Cross-Platform: Works on macOS, Linux, and Windows.
  • User-Friendly: Simple Gradio-based interface for accessibility.

Setup & Usage

  1. Install espeak-ng on macOS, Linux, or Windows.
  2. Create a Python 3.9+ virtual environment with uv or pip.
  3. Install dependencies via uv sync or pip install.
  4. Run python gradio_app.py and open http://127.0.0.1:7860 in a browser.
  5. Upload a PDF, select voices, choose API provider (OpenAI/OpenRouter/Kyutai), and generate audio.

Performance

Audio generation is parallelized across CPU cores. Performance depends on PDF length, available memory, CPU cores, and API network latency.

Troubleshooting

  • No audio: Ensure espeak-ng is installed.
  • API errors: Verify API key and endpoint.
  • PDF not processed: Ensure PDF contains readable text (not scanned images).
  • Memory issues: Large PDFs require more system memory.
