Overview

Whats-This-Rock is a Telegram bot that classifies images of rocks into one of seven types. I built this project using TensorFlow, with end-to-end capabilities including data processing, model training, and bot deployment. The project also uses nbdev for notebook-first development, documentation, packaging, and testing. [oai_citation:8‡GitHub](https://github.com/udaylunawat/Whats-this-rock)
