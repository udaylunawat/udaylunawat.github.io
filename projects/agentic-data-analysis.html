Agentic Data Analysis (Google ADK)

Agentic Google ADK Python Multi-Agent Orchestration Automation
View Repository

Agentic Data Analysis is a modular Python framework for autonomous data analysis, built using Google's Agent Development Kit (ADK). The codebase is organized around agent orchestration, enabling agents to perform data cleaning, statistical analysis, visualization, and reporting with minimal human intervention.

Repository Structure

agentic_data_analysis/
├── agents/
│   ├── data_cleaning_agent.py
│   ├── analysis_agent.py
│   └── visualization_agent.py
├── utils/
│   └── ...
├── config/
│   └── default.yaml
├── coordinator.py
└── main.py

How It Works

  1. main.py initializes the Coordinator and loads configuration.
  2. The Coordinator delegates tasks to agents (data cleaning, EDA, visualization).
  3. Agents communicate via defined interfaces, passing intermediate results and triggering downstream actions.
  4. Results are aggregated and a final report is generated, including visualizations and insights.

Key Features

  • Plug-and-play agent modules for extensibility.
  • Natural language query support (if enabled in config).
  • Automated EDA, visualization, and reporting.
  • Configurable workflows for different data analysis scenarios.

Example Usage

python main.py --config config/default.yaml --input data/sample.csv

Code Example

# main.py
from coordinator import Coordinator

if __name__ == "__main__":
    coordinator = Coordinator(config_path="config/default.yaml")
    coordinator.run(input_path="data/sample.csv")
